# Do I Know You?

Social game to test how well you know your friends. Built with React and Socket.IO.

### Features

- [x] Options selection
- [x] Random questions
- [x] Players as options
- [x] Open ended questions
- [x] Room host: Set time limit, kick players and transfer host
- [x] Rejoin game
- [x] Time limit

- [ ] Edit name
- [ ] SSM
- [ ] Room logs
- [ ] Admin page
- [ ] Projection mode
